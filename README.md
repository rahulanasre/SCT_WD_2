# ⏱️ Stopwatch Web Application

A simple and responsive Stopwatch Web Application built using HTML, CSS, and JavaScript.

- Start Stopwatch
- Pause Stopwatch
- Reset Stopwatch
- Record Lap Times
- Responsive Design
- User-Friendly Interface

- JavaScript
